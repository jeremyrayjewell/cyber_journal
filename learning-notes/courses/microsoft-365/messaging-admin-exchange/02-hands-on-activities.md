## Overview  

Section 2 explains how to practice with Microsoft 365 and Exchange Online safely. It focuses on using **simulations** and **trial subscriptions** so you can gain hands-on experience without risking a real production environment.

---

## Module 2.1: IMPORTANT – Using Assignments in the Course  

**Learning Objectives:**  
- Understand how assignments and simulations function in the course  

**Key Topics:**  
- Assignments consist of externally hosted, hands-on simulations   
- Assignment completion is **not required** for course completion or certificate eligibility  


---

## Module 2.2: DO NOT SKIP – Before Beginning Your Account Setup  

**Learning Objectives:**  
- Understand the requirements for setting up a Microsoft 365 lab environment  

**Key Topics:**  
- Course labs require a Microsoft 365 E5 subscription  
- Free access is obtained by starting an Office 365 E5 trial, then activating Microsoft 365 E5  

---

## Module 2.3: Creating a Trial Microsoft 365 / Azure Account  

**Learning Objectives:**  
- Understand how to create a safe Microsoft 365 lab environment for hands-on practice  
- Learn how to activate, assign, and later cancel a Microsoft 365 trial subscription  

**Key Topics:**  
- Creating a separate lab account prevents risk to production or business environments    
- Trial signup is accessed via (examlabpractice.com/microsoft365)[examlabpractice.com/microsoft365] to avoid broken links  
- Office 365 E5 trial is used to activate Microsoft 365 E5 features  
- Admin portal access has shifted from portal.microsoft.com to admin.microsoft.com  

—

## KEY TAKEAWAYS

- Microsoft 365 E5 features are accessed via an Office 365 E5 trial, then activated separately  
- Microsoft 365 admin portals and naming conventions change frequently; adaptability is required  
- Paid trials or alternative subscriptions (E3, Business Premium) may still support most course activities
