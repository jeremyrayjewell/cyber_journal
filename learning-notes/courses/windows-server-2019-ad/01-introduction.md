**Learning Objectives:**

- Install Windows Server 2019 in a virtualized lab environment  
- Perform basic post-installation setup for lab use  

**Key Topics:**

- Selecting a hypervisor (VirtualBox, VMware, Hyper-V)

## Module 1.3: Overview of Windows Server 2019  

**Learning Objectives:**  
- Understand the role of Windows Server 2019 in enterprise environments  
- Identify core server roles and features relevant to directory services and networking  
- Distinguish between common Windows Server editions and deployment models  

**Key Topics:**  
- Windows Server 2019 use cases in small to medium enterprise environments  
- Core server roles and features (AD DS, DNS, DHCP, File/Print, Group Policy)  
- Windows Server editions and licensing considerations (Standard vs Datacenter, evaluation licensing)  
- Server Core vs Desktop Experience trade-offs  
- Basic administration tools (Server Manager, MMC, PowerShell)  
- Patch management and update considerations in server environments

## KEY TAKEAWAYS

- Windows Server can be deployed effectively in a local virtual lab.  
- Core services (AD, DNS, GPO) run on top of the Windows Server platform.  
- Proper initial setup simplifies later domain and policy configuration.
