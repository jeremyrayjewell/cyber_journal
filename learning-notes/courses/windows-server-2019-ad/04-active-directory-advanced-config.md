## Overview  

Section 4 covers advanced Active Directory concepts, including administrative privileges, domain trusts, multi-domain management, and functional levels.

---

## Module 4.1: Domain and Enterprise Admins  

**Learning Objectives:**  
- Understand the roles of Domain Admins and Enterprise Admins  
- Apply least-privilege principles to privileged group membership  

**Key Topics:**  
- Purpose and scope of Domain Admins  
- Purpose and scope of Enterprise Admins  
- Privilege boundaries between domains and forests  
- Security risks of over-privileged accounts  
- Best practices for managing highly privileged groups  

---

## Module 4.2: Domain Trusts  

**Learning Objectives:**  
- Understand how trust relationships enable access across domains  
- Identify common trust types and their use cases  

**Key Topics:**  
- Purpose of domain trusts  
- One-way vs two-way trusts  
- Transitive vs non-transitive trusts  
- Trust relationships within forests and between forests  
- Security implications of trust configurations  
  
---

## Module 4.3: Changing Domains, DCs, and Exporting Data from AD  

**Learning Objectives:**  
- Navigate between domains and domain controllers in administrative tools  
- Export directory information for reporting or auditing  

**Key Topics:**  
- Switching domains and domain controllers in AD management tools  
- Viewing objects across multiple domains  
- Exporting AD data (e.g., users or groups) for documentation  
- Basic use cases for directory exports (inventory, audits)  
- Operational considerations when managing multi-domain environments  
  
---

## Module 4.4: Changing Functional Levels  

**Learning Objectives:**  
- Understand domain and forest functional levels  
- Safely raise functional levels after meeting prerequisites  

**Key Topics:**  
- Purpose of domain and forest functional levels  
- Prerequisites for raising functional levels  
- Compatibility considerations with older domain controllers  
- Benefits unlocked by higher functional levels  
- Risks and irreversibility considerations  
  
---

## KEY TAKEAWAYS

- Privileged groups require strict access control.  
- Trusts define access boundaries between domains.  
- Functional levels affect available AD features and compatibility.
